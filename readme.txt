== Description ==

Sends periodic e-mails for Gravity Forms entries created within the period. Daily, weekly, monthly udpates instead of 1 e-mail per form entry.

Plugin by [Weptile](http://weptile.com/ "WordPress Web Developers")


Change Log:
1.2.1
Added "delete schedule" feature
Added "schedule timestamp" feature (select what time of the day to send emails)

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the e-mail settings under Form Mail menu in the admin panel.
